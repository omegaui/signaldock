# Change Log

## 0.0.2

- Live transmission of terminal start/end events to connected SideKicks
- Bonjour/mDNS service advertisement (`_signaldock._tcp.`) for zero-config discovery
- Optional bridge password (`signalDock.bridgePassword`) for untrusted networks
- Workspace-info handshake so SideKick can label events per workspace
- LAN IP resolution via egress-route probe with interface-list fallback

## 0.0.1

- Receive notifications about long running terminal events
